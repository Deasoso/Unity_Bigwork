Information
'Simple 3D Tilemap Editor' is Voxel style Editor that use 3D Cube Object.

function
- Generate Grid  based 3D object Tilemap
- Json String based MapData File Save and Load

How to use MapGenerator
1.  Mouse right button click in hierachy
2.  Select'MapEditor/MapGenerator'
3.  Select MapGenerator Inspector
4.  Check 'IsDisplayGrid'
5.  Select(IsVertical) Grid Direction (Until completion must one direction Edit!)
6.  Set Value 'GridWorldSize' and 'NodeRadius'(Recommend GridWorldSize integer)
7.  Click LoadAsset
8.  Select your MapObjects(Prefab) Folder. (SampleProject MapObjects Path 'Resources/Enviroment')
9.  Select the objects button you want to Spawn.
10. Move your MousePointer to the Grid you want in SceneView
11. Edit Map(Reference bottom KeySetting)
12. If map is completed, Click SaveData,You want to Select Folder for Save (.dat)
13. Click LoadData ,Saved data can load

How to use CubeGenerator
1. Mouse right button click in hierachy
2. Select 'MapEditor/MapGenerator'
3. Select MapGenerator Inspector
4. Click 'CubeGenerator' Button
5. Click 'SaveFolder' Button to set the folder
6. Set Name,Size,Color
7. Click Create
8. Click the LoadAsset button in MapGenerator
9. Load the folder with the path set in step4
10.Select and edit the generated objects

KeySetting
- Spawn(E) 
- Remove(R) 
- MultiSelect(Ctrl + Click) 
- SelectOnlyGrid(Shift) 
- GridUp(Ctrl + ก่) 
- GridDown(Ctrl + ก้)
